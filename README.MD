Банковский калькулятор со слайдерами (ползунками) и управляемой SVG-диаграммой.
---
#### Установка зависимостей
$ npm i

#### Полная сборка
$ gulp

#### Сборка стилей
$ gulp styles

#### Сборка скриптов
$ gulp scripts
